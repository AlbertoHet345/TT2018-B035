\section{LR-IU2 Obtener zonas turísticas}

\subsection{Objetivo}
Permite visualizar las zonas turísticas que el turista ha visitado, de la misma manera, permite visualizar una zona turística actual, si es que el turista se encuentra en ella, para ello es necesario que se tenga habilitada la geolocalización del dispositivo móvil del turista.

\subsection{Diseño}
La pantalla \IUref{LR-IU2}{Zonas turísticas} muestra una lista con las zonas turísticas que el turista ha visitado, estas se encuentran ordenadas de mayor a menor puntuación. Así mismo, se muestra una zona turística actual, esta solo aparecerá si el turista, mediante la geolocalización de su smartphone, se encuentra dentro de ella.
