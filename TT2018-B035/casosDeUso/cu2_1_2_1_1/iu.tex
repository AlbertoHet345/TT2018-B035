\section{SE-IU2.1.2.1.1 Registrar Comentario}

\subsection{Objetivo}
Permite realizar el registro de un comentario y una calificación para un servicio turístico. Sin embargo, debe considerarse que sólo podrán registrar comentarios aquellos que cumplan la regla \cdtIdRef{BR-003}{Tiempo transcurrido para registrar comentario}.

\subsection{Diseño}
La pantalla \IUref{SE-IU2.1.2.1.1}{Registrar comentario} muestra el formulario para registrar un comentario al servicio turístico.
