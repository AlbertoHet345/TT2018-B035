\section{RA-IU3 Gestionar Zonas Turísticas}

\subsection{Objetivo}
Permite el llevar a cabo las operaciones necesarias para tener el control de las zonas turísticas del sistema.

\subsection{Diseño}
La pantalla \IUref{RA-IU3}{Gestionar Zonas Turísticas} muesta la tabla con las zonas turísticas que se tienen registradas en el sistema, cabe mencionar que la primera vez que se ingrese esta tabla se encontrará vacía. En esta pantalla el administrador podrá registrar zonas turísticas, editar y/o eliminar las que ya se encuentran registradas en el sistema.
